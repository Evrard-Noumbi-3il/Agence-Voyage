## Convention d'export

Pour chaque écran finalisé :
1. Exporter en PNG (1x et 2x) depuis Figma
2. Nommer : `{num}_{nom_ecran}_{plateforme}.png`
   Exemple : `01_connexion_mobile.png`
3. Placer dans ce dossier
